Lang Reader - User Guide

Welcome to Lang Reader, your powerful tool for language learning through video content. This guide will help you get started with the application and make the most of its features.

Note: For a visual demonstration of Lang Reader, check out our tutorial video.

Getting Started

Creating a New Project

1 Navigate to the Project List page using the navigation menu.
2 Click the "Create New Project" button.
3 Enter a name and description for your project.
4 Paste a YouTube URL or direct video link.
5 Click "Create Project" to save and begin.
Tip: Choose concise videos (5-15 minutes) for focused learning sessions.

Key Features

Creating Breakpoints

Breakpoints mark important moments in the video that you want to study:

  • During video playback, click the "Add Breakpoint" button at any moment you want to mark
  • Each breakpoint is recorded with its timestamp
  • You can add notes to each breakpoint to remind yourself why it's important

Loop Between Breakpoints

The looping feature allows you to repeatedly practice specific segments:

  • Select a start and end breakpoint from the dropdown menus
  • Set your desired loop count (how many times to repeat)
  • Set wait time between loops (in milliseconds)
  • Click "Start Breakpoint Loop" to begin the loop sequence
  • The system will automatically play the segment between your selected breakpoints the specified number of times

Project Notes

Keep track of your learning progress:

  • Use the notes section to write down your observations, vocabulary, grammar points, etc.
  • Notes are automatically saved with your project
  • Access your notes whenever you return to the project

Viewing and Managing Projects

Project List

The Project List page shows all your language learning projects:

  • Each project shows a thumbnail, title, description, and creation date
  • Click on a project thumbnail to quickly open it in view mode
  • Use the "View" button to open a project in read-only mode
  • Use the "Edit" button to make changes to a project
  • Use the "Delete" button to remove a project (this action cannot be undone)

Project View vs. Edit Mode

Lang Reader provides two ways to interact with your projects:

  • View Mode: For studying - you can play the video, use existing breakpoints, and run loops, but cannot add new breakpoints or change the project.
  • Edit Mode: For setting up - you can add new breakpoints, edit notes, and make changes to the project structure.

Tips for Effective Language Learning

  • Create Strategic Breakpoints: Mark moments where new vocabulary, idioms, or interesting pronunciation appears.
  • Use Looping Effectively: Start with short segments and gradually increase length as you build comprehension.
  • Practice Shadowing: Try to speak along with the video during loops to improve pronunciation.
  • Take Detailed Notes: Record new vocabulary, grammar patterns, and cultural insights in your project notes.
  • Regular Review: Return to previous projects periodically to reinforce learning.
